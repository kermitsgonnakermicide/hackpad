# The LTTPad 
This Hackpad was built as a fun little intro to the whole Highway thing - It contains:<br>
5x Keys<Br>
1x Encoder and<br>
1x 128x32 OLED Display<br>
This is powered by a Xiao RP2040 and the firmware will be KMK<br>
